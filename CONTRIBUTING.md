# Contributing

Thanks for taking a look at PowerPotion.

## Development

- Use Xcode 26.5 or newer.
- Keep changes scoped to the tool area they affect: `AnyPIP`, `DockMover`, or `WindowBuddy`.
- Do not commit local signing identities, provisioning profiles, derived data, archives, or distributable binaries.
- Run a local build before opening a pull request when possible.

## Pull Requests

Please include:

- What changed
- Why it changed
- Any manual testing performed
- Screenshots or recordings for visible UI changes
